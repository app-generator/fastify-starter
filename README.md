## AppSeed – coded apps

### Requirements

- Node.js — [installation instructions](https://nodejs.org/en/download/)
- Yarn — [installation instructions](https://yarnpkg.com/lang/en/docs/install/)

### Setup

From the project folder run:

```
	yarn install
```

### Run

From the project folder run:

```
	yarn server:start
```

Open link: http://localhost:5000
